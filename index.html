کاتالوگ نرم‌افزارهای سازمانی

🐳 زیرساخت اصلی

Docker

پلتفرم containerization - زیرساخت اصلی همه نرم‌افزارها. تضمین عملکرد یکسان در محیط‌های مختلف و سهولت استقرار و مدیریت.

Portainer

مدیریت Docker با رابط گرافیکی - ابزار قدرتمند برای مدیریت containerها، imageها و networkهای Docker از طریق رابط وب ساده و کاربرپسند.

📋 مدیریت پروژه و وظایف

Vikunja

مدیریت وظایف و پروژه - ابزار متن‌باز با رابط کاربری مدرن برای برنامه‌ریزی و پیگیری وظایف. قابلیت‌های تقویمی، برچسب‌گذاری و فیلترهای پیشرفته.

Focalboard

مدیریت پروژه بصری - جایگزین متن‌باز Trello با قابلیت‌های Kanban، مدیریت وظایف و پیگیری پیشرفت در محیط بصری و کاربرپسند.

Plane

جایگزین Jira - پلتفرم مدیریت پروژه پیشرفته با UI مدرن، مناسب برای تیم‌های نرم‌افزاری و مدیریت Agile.

Taiga

مدیریت پروژه Agile - ابزار ویژه تیم‌های چابک با پشتیبانی از Scrum، Kanban و مدیریت backlog.

OpenProject

مدیریت پروژه سازمانی - راه‌حل جامع برای مدیریت پروژه‌های بزرگ با قابلیت‌های گانت چارت، ردیابی زمان و مدیریت منابع.

Wekan

تخته Kanban ساده - ابزار سبک و سریع برای مدیریت وظایف به سبک Kanban با امکانات اساسی و کارآمد.

📄 مدیریت اسناد و امضای دیجیتال

DocuSeal

امضای دیجیتال اسناد - پلتفرم متن‌باز امضای دیجیتال با امنیت بالا، تایید هویت، ردیابی وضعیت اسناد و پشتیبانی از قالب‌های مختلف.

Paperless-ngx

مدیریت اسناد دیجیتال - سیستم هوشمند ذخیره‌سازی و سازمان‌دهی اسناد با OCR، جستجوی متنی قدرتمند و دسته‌بندی خودکار.

CryptPad

ویرایش مشترک رمزنگاری شده - پلتفرم امن برای ویرایش همزمان اسناد با رمزنگاری end-to-end و حفظ حریم خصوصی.

BookStack

ویکی و دانش‌پایه سازمانی - سیستم مدیریت دانش برای ایجاد و سازمان‌دهی مستندات، راهنماها و دانش‌پایه تیمی.

DokuWiki

مستندسازی تیمی - ویکی ساده و قدرتمند برای ایجاد و نگهداری مستندات فنی و سازمانی.

📊 تحلیل داده و هوش تجاری

Metabase

تحلیل داده و گزارش‌گیری - ابزار متن‌باز ایجاد داشبورد و تحلیل داده‌ها برای تیم‌های غیرفنی با رابط ساده و گزارش‌های بصری.

Apache Superset

پلتفرم BI پیشرفته - ابزار قدرتمند business intelligence با قابلیت‌های visualization پیشرفته و پشتیبانی از منابع داده متنوع.

Grafana

داشبورد و نمایش داده - پلتفرم پیشرفته برای ایجاد داشبوردهای زیبا و مانیتورینگ real-time با پشتیبانی از منابع داده مختلف.

PostHog

آنالیتیک محصول - پلتفرم جامع آنالیز رفتار کاربران، A/B testing و product analytics برای بهبود تجربه کاربری.

Plausible

آنالیتیک وب privacy-focused - جایگزین ساده و قابل اعتماد Google Analytics با تمرکز بر حفظ حریم خصوصی کاربران.

🔐 امنیت و مدیریت دسترسی

Passbolt

مدیریت رمز عبور تیمی - سیستم امن مدیریت و اشتراک‌گذاری رمزعبورها با رمزگذاری end-to-end و مدیریت دسترسی‌های دقیق.

Vaultwarden

سرور Bitwarden - پیاده‌سازی سبک سرور Bitwarden برای مدیریت شخصی رمزعبورها با همه قابلیت‌های premium.

Authelia

Single Sign-On و احراز هویت دومرحله‌ای - سیستم مدیریت دسترسی پیشرفته با پشتیبانی از 2FA، SSO و کنترل دسترسی دقیق.

Keycloak

مدیریت هویت سازمانی - پلتفرم جامع Identity and Access Management برای سازمان‌های بزرگ با پشتیبانی از پروتکل‌های مختلف.

LLDAP

سرور LDAP ساده - پیاده‌سازی سبک و ساده LDAP برای مدیریت کاربران و احراز هویت متمرکز.

💬 ارتباطات و همکاری

Mattermost

پلتفرم ارتباطات تیمی - جایگزین قدرتمند Slack برای ارتباطات امن تیمی با قابلیت میزبانی داخلی و یکپارچه‌سازی با DevOps tools.

Etherpad

ویرایش همزمان متن - ابزار ساده برای نوشتن و ویرایش مشترک اسناد در زمان واقعی با نمایش تغییرات لحظه‌ای.

Rocket.Chat

پلتفرم چت و همکاری - راه‌حل جامع ارتباطات تیمی با پشتیبانی از چت، ویدئو کال، فایل sharing و بات‌های هوشمند.

Element (Matrix)

پیام‌رسان غیرمتمرکز - پلتفرم ارتباطات امن و غیرمتمرکز با رمزگذاری end-to-end و قابلیت federation.

Jitsi Meet

ویدئو کنفرانس - راه‌حل متن‌باز برای جلسات ویدئویی و کنفرانس بدون نیاز به نصب اپلیکیشن.

BigBlueButton

وبینار و آموزش آنلاین - پلتفرم تخصصی برای کلاس‌های آنلاین، وبینارها و جلسات آموزشی با امکانات تعاملی.

🔧 توسعه و DevOps

n8n

اتوماسیون جریان کاری - پلتفرم قدرتمند اتوماسیون بدون کد با پشتیبانی از بیش از ۲۰۰ سرویس و امکان اجرای کدهای سفارشی.

GitLab

پلتفرم DevOps جامع - راه‌حل کامل برای مدیریت کد، CI/CD، مدیریت پروژه و مانیتورینگ در یک پلتفرم یکپارچه.

Gitea/Forgejo

Git hosting سبک - جایگزین سبک GitHub برای میزبانی مخازن Git با رابط ساده و عملکرد بالا.

Jenkins

CI/CD پیشرفته - ابزار قدرتمند اتوماسیون build، test و deployment با پشتیبانی از هزاران پلاگین.

📈 مانیتورینگ و نظارت

Uptime Kuma

مانیتورینگ uptime - ابزار ساده و زیبا برای نظارت بر دسترسی‌پذیری سرویس‌ها و وب‌سایت‌ها با alert‌های هوشمند.

Prometheus

مانیتورینگ پیشرفته - سیستم جامع جمع‌آوری متریک‌ها و هشدارهای هوشمند برای زیرساختها و اپلیکیشن‌ها.

Zabbix

مانیتورینگ شبکه و سرور - راه‌حل سازمانی برای نظارت بر زیرساخت‌های IT، شبکه و سرورها با قابلیت‌های پیشرفته.

LibreNMS

مدیریت و مانیتورینگ شبکه - ابزار تخصصی برای کشف خودکار و مانیتورینگ تجهیزات شبکه.

Netdata

مانیتورینگ real-time - داشبورد زنده عملکرد سیستم با نمایش آنی متریک‌های سیستم و اپلیکیشن‌ها.

🌐 مدیریت محتوا و وب

Nextcloud

پلتفرم ابری شخصی - جایگزین جامع Google Workspace برای ذخیره‌سازی فایل، تقویم، مخاطبین، ایمیل و همکاری تیمی.

Directus

Headless CMS پیشرفته - پلتفرم مدیریت محتوای بدون محدودیت با API خودکار و رابط کاربری قدرتمند برای مدیریت داده‌ها.

Strapi

CMS انعطاف‌پذیر - سیستم مدیریت محتوای headless با قابلیت سفارشی‌سازی بالا و API generation خودکار.

Ghost

پلتفرم انتشار محتوا - CMS مدرن برای بلاگ‌ها، خبرنامه‌ها و انتشار محتوای حرفه‌ای با طراحی زیبا.

vvveb

سازنده وب‌سایت بصری - ویرایشگر drag-and-drop برای طراحی وب‌سایت‌های واکنش‌گرا بدون نیاز به دانش برنامه‌نویسی.

LinkStack

مدیریت لینک‌های شخصی - ابزار ایجاد صفحات لینک شخصی‌سازی شده برای جمع‌آوری تمام لینک‌های مهم در یک مکان.

📞 پشتیبانی مشتری

FreeScout

مدیریت پشتیبانی مشتری - جایگزین ساده و کارآمد Zendesk برای مدیریت تیکت‌ها، پاسخ‌های آماده و پشتیبانی چندکاناله.

📝 نظرسنجی و بازخورد

Formbricks

نظرسنجی و جمع‌آوری بازخورد - پلتفرم متن‌باز ایجاد فرم‌های تعاملی، نظرسنجی‌ها و تحلیل بازخوردهای کاربران.

💼 مدیریت کسب‌وکار

ERPNext

سیستم ERP جامع - راه‌حل کامل مدیریت منابع سازمانی شامل حسابداری، CRM، موجودی، منابع انسانی و تولید.

Odoo

نرم‌افزار مدیریت کسب‌وکار - مجموعه ماژولار اپلیکیشن‌های تجاری شامل فروش، خرید، موجودی، تولید و مارکتینگ.

Invoice Ninja

مدیریت فاکتور و پرداخت - سیستم صدور فاکتور، مدیریت مشتریان، ردیابی پرداخت‌ها و گزارش‌گیری مالی.

Akaunting

حسابداری آنلاین - نرم‌افزار حسابداری ساده برای کسب‌وکارهای کوچک با قابلیت‌های فاکتور، گزارش‌گیری و مدیریت مالی.
