Portainer

Portainer یک ابزار قدرتمند و متن‌باز برای مدیریت Docker و Kubernetes است که به تیم‌های DevOps و توسعه‌دهندگان کمک می‌کند تا محیط‌های کانتینری را از طریق رابط کاربری گرافیکی زیبا و ساده مدیریت کنند. این پلتفرم جایگزین عالی برای مدیریت پیچیده‌های خط فرمان Docker است.

ویژگی‌های کلیدی

🐳 مدیریت Docker

کنترل کامل کانتینرها، تصاویر، شبکه‌ها و volumes با رابط بصری

☸️ پشتیبانی Kubernetes

مدیریت خوشه‌های Kubernetes، pods، services و deployments

🔐 کنترل دسترسی

RBAC، احراز هویت LDAP/OAuth و مدیریت کاربران چندگانه

📋 مدیریت Stacks

استقرار و مدیریت آسان استک‌ها با Docker Compose

📊 مانیتورینگ

نظارت real-time بر مصرف منابع و عملکرد کانتینرها

🔄 مدیریت Remote

کنترل چندین محیط Docker از یک پانل مرکزی

نصب و راه‌اندازی

نصب سریع Portainer:
# ایجاد volume برای ذخیره داده‌ها docker volume create portainer_data # اجرای Portainer docker run -d -p 8000:8000 -p 9443:9443 \ --name portainer --restart=always \ -v /var/run/docker.sock:/var/run/docker.sock \ -v portainer_data:/data \ portainer/portainer-ce:latest

پس از اجرا، به آدرس https://localhost:9443 مراجعه کنید

نصب با Docker Compose:
version: '3.8' services: portainer: image: portainer/portainer-ce:latest container_name: portainer restart: always ports: - "8000:8000" - "9443:9443" volumes: - /var/run/docker.sock:/var/run/docker.sock - portainer_data:/data security_opt: - no-new-privileges:true volumes: portainer_data:

⚠️ نکات امنیتی

  • رمز عبور اولیه: حتماً پس از اولین ورود، رمز عبور قوی تنظیم کنید
  • HTTPS: در محیط production از SSL certificate استفاده کنید
  • Docker Socket: دسترسی به Docker socket را محدود کنید
  • Network Security: از reverse proxy و firewall استفاده کنید
  • به‌روزرسانی: نسخه‌های جدید را به‌طور منظم نصب کنید
  • Backup: از volume داده‌ها بکاپ منظم تهیه کنید

پیکربندی پیشرفته

# استفاده از Let's Encrypt با Traefik version: '3.8' services: portainer: image: portainer/portainer-ce:latest labels: - "traefik.enable=true" - "traefik.http.routers.portainer.rule=Host(`portainer.yourdomain.com`)" - "traefik.http.routers.portainer.tls.certresolver=letsencrypt"

برای اتصال به Docker hosts از راه دور:

  1. در منوی Environments روی "Add environment" کلیک کنید
  2. نوع اتصال (Docker API, Agent) را انتخاب کنید
  3. آدرس و پورت سرور مقصد را وارد کنید
  4. در صورت نیاز، گواهی‌های TLS را تنظیم کنید

سوالات متداول

بله، Portainer برای محیط production مناسب است، اما باید نکات امنیتی را رعایت کنید. استفاده از HTTPS، تنظیم رمز قوی، و محدود کردن دسترسی‌ها ضروری است.

نسخه Community Edition (CE) رایگان است و برای اکثر کاربران کافی است. نسخه Business شامل پشتیبانی حرفه‌ای، RBAC پیشرفته، Git sync، و Edge computing capabilities است.

بله، Portainer از Kubernetes پشتیبانی می‌کند. می‌توانید خوشه‌های Kubernetes را اضافه کرده و pods، services، deployments و سایر منابع را مدیریت کنید.

# بکاپ از volume docker run --rm -v portainer_data:/data -v $(pwd):/backup alpine tar czf /backup/portainer-backup.tar.gz -C /data . # بازیابی docker run --rm -v portainer_data:/data -v $(pwd):/backup alpine tar xzf /backup/portainer-backup.tar.gz -C /data
